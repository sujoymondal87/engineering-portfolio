# Engineering Portfolio & Case Study Platform

A full-stack portfolio platform built to document production architecture, 
runtime systems, platform evolution case studies and engineering decisions 
from 15+ years of software development.

**Live:** https://sujoymondal-tech.vercel.app

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: PostgreSQL, Supabase
- Deployment: Vercel (frontend), Railway (backend)
- Auth: Supabase Auth
- Storage: Supabase Storage

## Features
- Case study publishing with markdown support
- Admin panel with protected routes
- Image upload to Supabase Storage
- SEO optimization — meta tags, Open Graph, Twitter Card, JSON-LD structured data, sitemap.xml, robots.txt, Google Search Console indexed
- Mobile responsive
- Git-based CI/CD workflow
